## 1. 模块概览

TikTok 数据管理模块共包含三个核心功能页面，用于管理从 TikTok 平台采集到的主播及观众数据，支持筛选、导出、数据分配和主播画像分析。

| 页面 | 功能定位 |
|------|---------|
| **采集主播** | 查看、筛选、导出已采集的 TikTok 直播间主播数据 |
| **大哥数据** | 查看直播间高贡献值观众（送礼榜单）数据 |
| **可邀约主播** | 管理可邀约合作的主播，支持数据分配和画像分析 |

## 2. 采集主播

### 2.1 页面介绍

采集主播页面展示从 TikTok 直播间采集到的主播基本信息，包括账号、昵称、直播间数据及导出状态。

![home-qntw.png](https://mirrorseek.com/upload/home-qntw.png)

---

### 2.2 数据筛选

点击搜索栏上方的【展开/收起】可切换更多筛选条件。

| 筛选项 | 说明 |
|--------|------|
| 采集时间 | 按数据采集的时间范围筛选 |
| TikTok账号 | 精确或模糊匹配主播的 TikTok 账号（displayId） |
| 主播昵称 | 按昵称关键词搜索 |
| 直播间ID | 输入直播间 ID 精确查询 |
| 用户ID | 输入主播内部用户 ID 精确查询 |
| 话题分类 | 按话题标签/分区筛选 |
| 在线人数区间 | 设置最小/最大值，筛选在线人数范围 |
| 粉丝区间 | 按粉丝数范围筛选 |
| 钻石数区间 | 按钻石总数范围筛选 |
| 建联状态 | 未建联 / 已建联 |
| 导出状态 | 未导出 / 已导出 |

![anchor_search.png](/upload/anchor_search.png)

---

### 2.3 导出数据

1. 根据需要设置筛选条件，缩小导出范围
2. 点击右上角【导出】按钮
3. 系统自动下载 Excel 文件
4. **导出完成后，被导出的记录状态自动更新为「已导出」**，列表实时刷新

![anchor_export.png](/upload/anchor_export.png)

---

### 2.4 建联状态管理

在表格操作列中，可以对每条主播记录单独修改建联状态：

- 点击【建联】→ 状态变为「已建联」（绿色标签）
- 点击【取消建联】→ 状态变为「未建联」（灰色标签）

![anchor_contact.png](/upload/anchor_contact.png)

## 4. 可邀约主播

### 4.1 页面介绍

可邀约主播是经过筛选后进入邀约流程的目标主播列表。此页面支持**数据分配**（将主播分配给指定账号管理）、**画像分析**（查看主播详细数据）等高级功能。

![invite_dashboard.png](/upload/invite_dashboard.png)

---

### 4.2 数据筛选

| 筛选项 | 说明 |
|--------|------|
| 筛选入库时间 | 按主播入库时间范围筛选 |
| TikTok账号 | 精确或模糊匹配账号 |
| 主播昵称 | 按昵称关键词搜索 |
| 直播间ID | 精确查询 |
| 用户ID | 精确查询主播内部 ID |
| 地区 | 按国家/区域代码筛选 |
| 邀约类型 | 按邀约分类筛选 |
| 分配状态 | 未分配 / 已分配 |
| 导出状态 | 未导出 / 已导出 |
| 建联状态 | 未建联 / 已建联 |

![invite_search.png](/upload/invite_search.png)

---

### 4.3 数值区间筛选

展开搜索栏后，可以对以下 5 个数值字段设置「最小值—最大值」区间进行精准筛选：

| 字段 | 用途示例 |
|------|---------|
| 在线人数 | 筛选直播间有一定规模的主播，如在线人数 ≥ 1000 |
| 粉丝数 | 筛选中腰部及以上主播，如粉丝 ≥ 10万 |
| 关注数 | 辅助判断账号活跃度 |
| 总点赞数 | 评估内容影响力 |
| 钻石总数 | 衡量直播间变现能力 |

**操作方式：** 在对应区间输入框中填入最小值或最大值（可只填其中一个），点击【搜索】即可生效。

![invite_filter.png](/upload/invite_filter.png)

### 4.4 数据分配

数据分配功能允许管理员将指定主播分配给某个账号，被分配的账号登录后**只能看到分配给自己的数据**。

#### 分配操作步骤

1. 在列表中勾选需要分配的主播（支持跨页多选）
2. 点击顶部工具栏的【分配数据】按钮
3. 弹出分配对话框，左侧为已选主播列表，右侧选择目标用户

![invite_data_assignment_user.png](/upload/invite_data_assignment.png)

4. 在右侧下拉框中选择目标用户
5. 点击【确认分配】完成操作

![invite_data_assignment_success.png](/upload/invite_data_assignment_success.png)

#### 分配规则说明

> ⚠️ **注意事项**
>
> - 每条主播数据**只能分配给一个用户**，不支持重复分配
> - 若所选主播中包含已被分配的记录，系统会**自动跳过**并在提示中告知跳过数量
> - 分配完成后，「分配状态」列将显示被分配用户的名字

![invite_data_assignment_user.png](/upload/invite_data_assignment_user.png)


#### 取消分配

在操作列中点击【取消分配】，可解除该主播的分配关系，状态恢复为「未分配」。

## 5. 主播画像分析

### 5.1 进入画像页面

在**可邀约主播**列表中，点击「用户ID」列中的蓝色链接文字，即可跳转至该主播的画像分析页面。

![invite_click_userid.png](/upload/invite_click_userid.png)

---

### 5.2 核心数据指标

画像页面顶部展示主播基本信息卡片，中部展示 5 个核心运营指标：

| 指标 | 说明 |
|------|------|
| 在线人数 | 数据采集时直播间的实时在线观众数 |
| 粉丝数 | 主播账号总粉丝数量 |
| 关注数 | 主播关注的账号数量（辅助判断账号真实性） |
| 直播间点赞 | 本场直播获得的总点赞数 |
| 钻石总数 | 直播间累计获得的钻石（虚拟礼物换算） |

![invite_profile_analyze.png](/upload/invite_profile_analyze.png)

在主播信息卡片中，**点击主播昵称**可在新标签页中直接打开该主播的 TikTok 主页。

![invite_profile_click_userid.png](/upload/invite_profile_click_userid.png)

---

### 5.3 高质量观众 TOP 10

页面下方展示该主播直播间**贡献值最高的前 10 名观众**，通过横向进度条直观呈现各观众的贡献值占比。

| 字段 | 说明 |
|------|------|
| 排名 | 金/银/铜徽章标注前三名 |
| 昵称 | 观众账号昵称 |
| TikTok账号 | 观众的 @账号 |
| 贡献值 | 观众在该直播间的礼物贡献分数 |
| 粉丝数 / 关注数 | 辅助评估观众账号质量 |

![invite_profile_analyze_top10.png](/upload/invite_profile_analyze_top10.png)

**点击观众昵称**同样可在新标签页打开该观众的 TikTok 主页，方便快速了解观众背景。

![invite_profile_click_audience_id.png](/upload/invite_profile_click_audience_id.png)
